Spire

Spire is powerful math library for Scala. Going beyond the basic functionality provided by Scala and Java, Spire aims to provide powerful new number types, type classes for generic math and algebra, and other innovations to make working with numbers in Scala even better.

Key features

  1. New number types:
    • Rational
    • Complex[A]
    • Real
    • and more...
  2. Specialized type classes like Ring, Integral and Numeric
  3. Expanded set of operators and functions for all number types
  4. Compile-time checking/optimization for numeric constants

Getting Spire

Spire is available to you under an MIT license, and can be downloaded from the Sonatype OSS repo. To add Spire to an SBT project add the following lines to your build.sbt file:


scalaVersion := "2.10.0-M6"

resolvers += "sonatype" at "https://oss.sonatype.org/content/repositories/releases"

libraryDependencies += "org.spire-math" %% "spire" % "0.3.0-M1"

(Currently Spire is only available as a milestone against Scala-2.10.0-M6.)

You can also visit the project page.

Using Spire


import scala.{specialized => sp}
import spire.math._
import Implicits._

// example of a generic distance function
def distance[@sp A:Numeric](x:A, y:A) = Numeric[A].sqrt(x * x + y * y)
