Spire

Spire is a powerful math library for Scala. Going beyond the basic functionality provided by Scala and Java, Spire aims to provide powerful new number types, type classes for generic math and algebra, and other innovations to make working with numbers in Scala even better.

Key features

  1. New number types:
    • Rational
    • Complex[T]
    • Real
    • and more...
  2. Specialized type classes like Ring, Integral and Numeric
  3. Expanded set of operators and functions for all number types
  4. Compile-time checking/optimization for numeric constants

Getting Spire

Spire is available to you under an MIT license, and is kindly hosted by the Sonatype OSS repo. The current version of Spire is being developed against Scala 2.10. To add Spire to an SBT project add the following lines to your build.sbt file:


scalaVersion := "2.10.0-RC1"

libraryDependencies += "org.spire-math" % "spire_2.10.0-RC1" % "0.3.0-M3"

You can also visit the project page.

Using Spire


import scala.{specialized => sp}
import spire.math._
import spire.implicits._

// example of Euclidean distance function
def distance[@sp A: Numeric](x: A, y: A): A =
  (x * x + y * y).sqrt

// greatest common denominator
def gcd[@sp A: Integral](x: A, y: A): A =
  if (x % y == 0) y else gcd(y, x % y)
